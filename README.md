# Case4
Программа, которая принимает на вход файл с данными о пользователях и их друзьях в одной из социальных сетей,
выводит на экран количество пользователей, у которых больше 100 друзей. Кроме того, программа должна строить круговую диаграмму,
показывающую процентное соотношение пользователей по количеству друзей: менее 10, от 10 до 50, от 50 до 100 и более 100.
